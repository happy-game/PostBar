Postlight的Headless WordPress + React Starter Kit是GithHub上一个开源的自动化工具集：支持一步到位启动一个由WordPress驱动的React应用程序。项目地址：https://github.com/postlight/headless-wp-starter。 官方安装步骤要求用户要先下载Docker，这里提供一个不用下载安装Docker，小白也能操作使用该开源工具集的方法。

先介绍下实现这一过程的工具： TeamCode Tin是一个能够快速从源码构建可运行实例的工具。使用界面完全跳过底层操作，技术小白也能完全看的懂操作。用户使用Tin应用即可直接在云端运行Headless WordPress。

操作步骤：

  1. 点击Headless WordPress的Tin链接 https://www.teamcode.com/tin/clone?applicationId=273434815208255488
  2. 点击后进入TeamCode登录页面完成登录（已登录用户可跳过该步骤）。你可以选择邮件、手机号、google账户、微信以及QQ进行快速登录。 注册TeamCode的用户，每个月都有一定的免费额度可以使用。
  3. 点击 Clone，将Headless WordPress克隆到自己的Tin工作空间。
  4. 点击 RUN NOW，Tin应用会立即运行。
  5. 点击Primary Endpoint，即可立即访问Headless WordPress。
  6. 你已经可以开始使用WordPress。
  7. 你可以在Tin的概览页自定义运行时硬件配置，以满足你对运行效果的不同需求。

如果你想同样快速的安装使用其他开源项目，跳过环境安装配置过程，可以在TeamCode官网查看Tin的详细使用指南 https://www.teamcode.com/docs/tin/create-tin，更多好用的功能等你发现。
