<div class=“markdown-text”><p>一年一度的双十一购物狂欢节刚轰轰烈结束，万千好物装进一个小小的App，只需手指滑动页面，足不出户就能尽揽世界美好。不仅仅是购物，一键出行，一键吃喝玩乐，技术发展催生的各项服务让生活越来越便捷。而究其背后，我们的美好生活的广厦离不开Web技术的添砖加瓦。</p> <h2>Web技术的现在和未来式</h2> <p>近年来Web技术得到了飞速发展，特别是各种前端应用层出不穷。Web与生俱来的可链接性、可检索性和沉淀数十年的能力产生了各种各样的服务，我们的生活与Web技术的发展息息相关。而Web相比原生应用体验不完善的地方，也催生了渐进式网页应用(Progressive Web Application)技术的诞生。</p> <blockquote> <p><strong>PWA小课堂</strong> 渐进式网页应用 (Progressive Web Apps) 简称为 PWA。是由谷歌提出推广的，在多平台下利用现代的 Web API 提供的标准化框架，使得网页可以像传统应用或原生移动应用一样运作，以及以传统的渐进式增强策略来创建跨平台 Web 应用程序，使其具有与原生应用相同的用户体验优势。 <img src=“https://img.alicdn.com/imgextra/i3/O1CN01934Cfx27roEg259qj_!!6000000007851-1-tps-597-721.gif” alt=“image.png”></p> </blockquote> <p>PWA技术可谓是未来可期，随着“瞬时应用安装”，“离线工作模式”，“本地文件访问”，“高性能WebAssembly”等新能力逐步引入，PWA正在技术演进的路上狂飙。</p> <h2>中国PWA开发者日上线！</h2> <p>为加速推动PWA在中国的落地和发展，Intel与鼎道智联强强联手，携手打造“中国PWA开发者日”，并将于11月27日在线上举行大会。本次活动以“Web开发的现在与未来”为主题，并邀请一众业界顶级大咖围绕PWA技术展开探讨，分享技术进展以及不同领域的融合。</p> <p><img src="//static.cnodejs.org/FpM7MsGPn9kMsb8TXyM-QqeqT6TF" alt=“image.png”> 大会主办方Intel与鼎道智联对本次大会也是期待值拉满：</p> <p><img src="//static.cnodejs.org/FokjeuldFlMxYxZofQRDqUpQiEFO" alt=“image.png”></p> <h2>业界大咖云集，干货答疑尽揽</h2> <p>本次活动还将邀请国内知名的Web技术专家就Web技术的最新进展以及PWA在产品中落地的成功案例来展开讨论，专注于基于PWA技术的Web前沿应用，聚焦PWA在英特尔、鼎道智联、阿里巴巴、腾讯、谷歌 等热点公司的应用实例进行深入分享。同时将安排互动十足的Q&A环节，让大咖在线答疑解惑，观点争锋，以期与开发者共同推动PWA技术的未来，引领行业发展。</p> <p>前方大咖来袭（持续更新中），先睹为快吧！ <img src="//static.cnodejs.org/Fm6TapyqQKhTqls8ChUgegkMj8oT" alt=“Xnip2021-11-24_17-01-11.png”> <img src="//static.cnodejs.org/FiGgdAXArb9Hx4Dzzu0NmNPsMqVl" alt=“Xnip2021-11-24_17-01-39.png”></p> <p>精彩内容不停歇，本次PWA开发者日将会持续给开发者带来更多的分析和干货，感兴趣的小伙伴们快快扫下方二维码参与活动，关注获得会议议程，第一时间掌握PWA开发者日前沿动态！ <img src="//static.cnodejs.org/FklhdJ3bBkIVmEW02y8nEZtz5OXN" alt=“Xnip2021-11-24_17-02-32.png”></p> <p>戳<a href=“https://mp.weixin.qq.com/s?__biz=MzU3MDg5NTI0OQ==&mid=2247488754&idx=1&sn=0fcb7c8cc245624ea975e6c0524bc7ac&chksm=fce92a0ccb9ea31a7d7d385a0e3bbb732ac9934c34d2ebdd931f7669297888978f72d652173f##”>此链接</a>在线报名～</p> </div>
